# 1688-Crawler

Based on Flask, require python3.6

## Install

Create a virtual environment by venv

```sh
$ python3 -m venv venv
$ source ./venv/bin/activate
```

Install requirements by pip:

```sh
$ pip install -r requirements.txt
```

## Run

```python
python run.py
```

Open http://127.0.0.1:5000 in a browser.
